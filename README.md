# Statistical_Laboratory_R_Code_Samples
Coding excerpts in R completed for ST121 Coursework

## Assignment 1
### Overview
1) Demonstrate use of basic R commands with numbers and vectors.
2) Answer a series of short mathematical questions:
   a)  Write a function that determines if a number is prime.
   b) Write a function that determines if a number is a square number. Use it to determine the number of square numbers between 1000 and 10000.
3) Answer a series of long mathematical questions:
   a) Given coordinates of the vertices, write a function that determines the area of a triangle.
   b) Create a function that computes the area of all eight "elementary triangles" within the unit square.

### Files:
*Assignment 1.pdf*: Official assignment instructions.
*Assignment 1_Response.pdf* : Completed assignment document.
